# Hasm - Hack Assembler

My implementation of the Hack assembler described in Nand 2 Tetris and their
accompanying book "The Elements of Computing Systems, Building a Modern Computer
from First Principles" by Noam Nisan and Shimon Schocken

## Prerequisites

- uv
- Docker (optional)

## Quick Start

```console
$ uv sync
$ hasm <file>
```

Will output assembled hack programs in current directory as text files with instructions line separated
